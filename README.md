# eduAvança

O que é esse projeto?
-

Baseado na premissa da ODS 4.2. Aqui apresenta-se um protótipo de uma plataforma de microcursos para capacitar educadores da primeira infância e uma plataforma auxiliar de pseudo-fórum onde gestores compartilhariam boas práticas que dão certo em suas creches para serem votadas e auxiliar no melhoramento continuo da plataforma.

Tecnologias Usadas
